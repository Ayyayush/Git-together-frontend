## ğŸ”‘ Step 3: Key Pair (Very Important)

When creating a server, we need a secure way to access it.

- AWS provides a `.pem` file
- `.pem` file is the private key of the server
- It acts like a password for SSH access
- Without it, the server cannot be accessed

Rule:
- Never share the `.pem` file
- Never upload it to GitHub
